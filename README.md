# Atividade Pokedex 👋

Esse é um projeto criado com [Expo](https://expo.dev) 

## iniciando

1. Baixe as depências 

   ```bash
   npm install
   ```

2. Inicie o projeto

   ```bash
   npx expo start
   ```

## Importante
### Dados do Perfil
Utilize uma conta que ja existe ou crie uma nova

### Arquivo Postman
é necessario o arquivo json postman instalado para o projeto funcionar
